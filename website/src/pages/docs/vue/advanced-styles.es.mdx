# Estilos avanzados

En el caso de Vue utilizamos clases para dar estilo a los elementos.

- `.le-aspect-ratio`: el elemento `<div>` que funciona como contenedor.
- `.le-aspect-ratio-item`: el elemento hijo de `.le-aspect-ratio`.
- `.le-vi-iframe`: el elemento `<iframe>` dentro del elemento `<vimeo-lite>`.
- `.le-vi-play-button`: el elemento `<button>` dentro del elemento `<vimeo-lite>`.
- `.le-vi-play-icon`: el elemento `<svg>` dentro del elemento `<button>` de `<vimeo-lite>`.
- `.le-yt-button`: el elemento `<button>` dentro del elemento `<youtube-lite>`.
- `.le-yt-icon`: el elemento `<svg>` dentro del elemento `<button>` de `<youtube-lite>`.
- `.le-yt-iframe`: el elemento `<iframe>` dentro del elemento `<youtube-lite>`.
