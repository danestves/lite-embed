# Advanced Styles

For Vue we use classess to style elements.

- `.le-aspect-ratio`: the `<div>` element that works as a container.
- `.le-aspect-ratio-item`: the child element of `.le-aspect-ratio`.
- `.le-vi-iframe`: the `<iframe>` element inside the `<vimeo-lite>` element.
- `.le-vi-play-button`: the `<button>` element inside the `<vimeo-lite>` element.
- `.le-vi-play-icon`: the `<svg>` element inside the `<button>` element of `<vimeo-lite>`.
- `.le-yt-button`: the `<button>` element inside the `<youtube-lite>` element.
- `.le-yt-icon`: the `<svg>` element inside the `<button>` element of `<youtube-lite>`.
- `.le-yt-iframe`: the `<iframe>` element inside the `<youtube-lite>` element.
