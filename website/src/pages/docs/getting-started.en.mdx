# Getting Started

## Installation

### npm

```bash
# react
npm add @lite-embed/react

# vue
npm add @lite-embed/vue
```

### yarn

```bash
# react
yarn add @lite-embed/react

# vue
yarn add @lite-embed/vue
```

### pnpm

```bash
# react
pnpm add @lite-embed/react

# vue
pnpm add @lite-embed/vue
```

## Quick Start

Here is a quick snippet to get you started with Lite Embed when using React.

```tsx
// Dependencies
import { YoutubeLite } from '@lite-embed/react';

function App() {
  return (
    <YoutubeLite
      urlOrId="https://www.youtube.com/watch?v=dQw4w9WgXcQ" // or dQw4w9WgXcQ
    />
  );
}
```

That will result on the following:

<div>
  <YoutubeLite urlOrId="https://www.youtube.com/watch?v=dQw4w9WgXcQ" />
</div>
