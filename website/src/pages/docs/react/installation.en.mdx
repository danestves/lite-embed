# Installation

You can install `@lite-embed/react` with npm, yarn or any package manager. But also if you are using Vue, you can use the component with an `<script />` tag via [unpkg.com](https://unpkg.com).

## npm

```bash
npm install @lite-embed/react
# or
yarn add @lite-embed/react
```

`@lite-embed/react` is compatible with React v16.8+.
