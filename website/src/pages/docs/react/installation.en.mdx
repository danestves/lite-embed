# Installation

You can install `@lite-embed/react` with [npm](https://www.npmjs.com/@lite-embed/react), [yarn](https://yarnpkg.com/package/@lite-embed/react) or any package manager.

## npm

```bash
npm install @lite-embed/react
# or
yarn add @lite-embed/react
```

`@lite-embed/react` is compatible with React v16.8+.
