# Installation

You can install `@lite-embed/react` with [npm](https://npmjs.com), [yarn](https://yarnpkg.com/) or any package manager.

## npm

```bash
npm install @lite-embed/react
# or
yarn add @lite-embed/react
```

`@lite-embed/react` is compatible with React v16.8+.
