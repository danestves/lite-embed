# Installation

You can install `lite-embed` with npm, yarn or any package manager. But also if you are using Vue, you can use the component with an `<script />` tag via [unpkg.com](https://unpkg.com).

## npm

```bash
# For React:
npm install @lite-embed/react

# For Vue:
npm install @lite-embed/vue
```

`lite-embed` is compatible with React v16, v17 and v18.

<Callout emoji="ðŸ’¡">
  For Vue we currently only support Vue v3 and above.
</Callout>

## CDN

If you want to use our `UMD` version of `lite-embed` in your project, you can use the following `<script />` from [unpkg.com](https://unpkg.com):

```html
<script src="https://unpkg.com/@lite-embed/vue/dist/lite-embed-vue.umd.js"></script>
```
